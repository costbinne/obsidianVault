## ① 核心网络（先记这4个）

#VPC   = Virtual Private Cloud        → 私有网络  
#CIDR  = Classless Inter-Domain Routing → IP范围（10.0.0.0/16）  
Classless  
= 不使用传统IP分类（A/B/C类）
Inter-Domain  
= 网络之间
Routing  
= 路由
#Subnet = 子网                        → VPC里切一块  
#IP    = Internet Protocol           → 地址

---

## ② 结构层级（空间关系）

#Region = 区域（东京）  
#AZ     = Availability Zone → 数据中心

👉 关系（必须记）：

Region  
 └─ VPC（跨AZ）  
     └─ Subnet（属于某个AZ）

---

## ③ 出网 & 路由（高频考点）

#RouteTable  #Route = 路由规则（目标IP → 去哪）  
#IGW         = Internet Gateway → 直接上网  
#NAT         = Network Address Translation → 私网出网

👉 核心记忆：

0.0.0.0/0 → 默认出口

---

## ④ 安全（一定会考）

#SG   = Security Group → 实例防火墙（有状态）  
#NACL = Network ACL    → 子网防火墙（无状态）

👉 对比一句话：

#SG = allow + stateful  
#NACL = allow/deny + stateless

---

## ⑤ 外部连接（重点）

#VPN   = Virtual Private Network → 加密隧道  
#IPsec = Internet Protocol Security → VPN加密协议  
#VGW   = Virtual Private Gateway → AWS侧VPN入口  
#DX    = Direct Connect → 专线

---

## ⑥ 用户 & 企业接入

#ClientVPN      → 个人 → VPC  
#Site-to-Site   → 公司 → VPC

---

## ⑦ 多网络架构（进阶）

#VPCPeering  → VPC ↔ VPC（点对点）  
#TGW         → Transit Gateway（中心枢纽）

---
