#### 后端 / Web

- 框架：Spring Boot
    
- 服务器：外置 Tomcat
    
- 打包方式：WAR
    
- 前端模板：Thymeleaf
    

---

#### 构建 / 依赖 / 版本管理

- 构建工具 & 依赖管理：Maven
    
- 版本管理：Git
    

---

#### 数据库

- DB 本体：MySQL
    
- 数据库连接：JDBC
    
- ORM / 数据访问层：MyBatis
    
- Schema 版本管理：Flyway
    

---

#### 测试 / 质量管理

- 单元测试：JUnit
    
- Maven 单测插件：maven-surefire-plugin
    
- 覆盖率：JaCoCo
    
- 静态检查：Checkstyle 或 SpotBugs（任选其一）
    

---

#### 配置 / 安全 / 日志

- 配置管理：
    
    - application.yml
        
    - application-dev.yml
        
    - application-prod.yml
        
- 日志：Logback（Tomcat 外置环境）
    
- 安全：Spring Security（Basic Auth 或 Session）
    

---

#### 文档

- API 文档：OpenAPI（springdoc-openapi）
    

---

#### CI / CD

- CI：GitHub Actions
    
    - 单元测试
        
    - 构建 WAR
        
- CD：AWS
    
    - EC2
        
    - systemd 管理 Tomcat
        

---

#### 服务器环境

- OS：Linux
    
- Java：JDK（与 Tomcat 版本匹配）
